# Cristiano Ronaldo Project

This is a web project about the football legend Cristiano Ronaldo. I made this website to show my skills in HTML and CSS and to talk about the history of CR7.

## Inside the website:
- Personal story about Ronaldo and his career.
- Statistics table showing his performance with clubs and national team.
- A long list of all his major trophies like Champions League and league titles.
- Individual achievements table like the Ballon d Or and Golden Boots.
- Images showing his best moments in football.

## How I built it:
I used basic HTML tags to organize the page and tables to display the statistics clearly. I styled the page with a dark background and gold text using CSS. I also added some simple hover animations to make the tables look better when you move the mouse.
